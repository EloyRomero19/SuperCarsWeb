# 🚗 SuperCarsWeb

## 📖 Overview

SuperCarsWeb is a modern web application developed with Angular that allows users to explore a collection of supercars, read automotive news and navigate through different sections of the website using Angular Routing.

This project was developed as part of the Higher Vocational Training in Web Application Development (DAW).

---

## ✨ Features

- Home page
- Supercars catalogue
- Automotive news section
- News detail page
- Contact page
- About us section
- Responsive navigation bar
- Custom 404 page
- Responsive design

---

## 🛠 Technologies

- Angular 19
- TypeScript
- HTML5
- CSS3
- Bootstrap
- Angular Router
- Angular Forms
- Express (Angular SSR)

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/EloyRomero19/SuperCarsWeb.git
```

Install dependencies

```bash
npm install
```

Run the application

```bash
ng serve
```

Open

```
http://localhost:4200
```
